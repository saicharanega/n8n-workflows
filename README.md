# 🔄 n8n Automation Workflows Portfolio

This repository contains my automation workflows built using n8n (workflow automation tool).
These workflows demonstrate my skills in API integrations, AI automation, agents, and no-code/low-code system design.

---

## 🚀 Workflows Included

### 📰 AI News Summarizer Automation
Automates daily AI news collection from multiple sources, summarizes them using Google Gemini AI, and sends a formatted newsletter via Gmail.

**Features:**
- RSS feed aggregation
- Event scraping via SerpAPI
- AI-powered summarization
- Automated email delivery

**Tech Used:** n8n, Google Gemini AI, Gmail API, SerpAPI

---

### 🎓 AI Learning Path Generator Agent
An intelligent AI agent that creates personalized learning roadmaps.
It generates structured curricula, creates Google Docs, and schedules study plans in Google Calendar automatically.

**Features:**
- AI Agent with tools
- Google Docs automation
- Calendar event scheduling
- Resource research via APIs

**Tech Used:** n8n, AI Agents, Google Docs API, Google Calendar API, Gemini AI
